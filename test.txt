Pasos para subir un archivo o cambio a repo github

# Menú → File → Open Folder → selecciona tu carpeta

 #Abre la terminal integrada en VS Code (Ctrl + ñ o Ctrl + J y luego selecciona terminal).

    git init

# Verifica el remoto (el enlace de tu repo GitHub):

        git remote add origin https://github.com/hellersj/tf003.git

# Si ya existe y quieres actualizarlo:

    git remote set-url origin https://github.com/hellersj/tf003.git

# Agrega tu archivo:

    git add test.txt

# Haz un commit con mensaje:

    git commit -m "Agrego archivo test.txt"

# Sube el commit a GitHub (rama main):

    git branch -M main
    git push -u origin main

# Desde VS Code también puedes hacerlo con la interfaz gráfica:

En la barra lateral izquierda, haz clic en el icono de Source Control (Control de código fuente).
Verás test.txt como archivo modificado → haz clic en el + para Stage.
Escribe un mensaje en el campo de commit y presiona el ✔️.
Luego, en la parte superior, dale clic en “Publish Branch” o “Push”.
